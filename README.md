# Executive Workforce Insights Dashboard

## Project Overview
This Power BI dashboard provides an executive-level overview of workforce metrics, enabling HR and management to monitor workforce composition, employee demographics, and attrition trends through interactive visualizations.

## Objectives
The main objectives of this project are:
- To provide a comprehensive overview of workforce metrics and HR performance.
- To analyze workforce demographics, department distribution, and attrition trends.
- To enable data-driven decision-making through interactive dashboards and visual insights.

## Dashboard Features
- Overall Headcount
- Department Distribution
- Gender Ratio
- Age Group Analysis
- Years of Service
- Job Grade Distribution
- Attrition Analytics
- Interactive Department Filter

## Disclaimer
This dashboard is shared for portfolio purposes. Any sensitive information has been anonymized or is used solely for demonstration.
